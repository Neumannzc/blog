## 一、截图工具

flameshot

Ubuntu 22.04+ 默认使用Wayland 显示协议，而 Flameshot 的旧版本仅支持 X11 协议，导致截图异常。

编辑配置文件：

```bash
sudo nano /etc/gdm3/custom.conf
```

找到行 #WaylandEnable=false，删除 #取消注释，改为：

```bash
WaylandEnable=false
```
