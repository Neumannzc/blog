## 一、查看宝塔入口

`sudo /etc/init.d/bt default | grep 外网面板地址`

## 二、查看账号

`sudo /etc/init.d/bt default | grep username`

## 三、查看密码

`sudo /etc/init.d/bt default | grep password`

## 四、查看宝塔是否启动

`systemctl status bt`

## 五、启动宝塔

`systemctl start bt`

## 六、重启宝塔

`systemctl restart bt`

## 七、重置密码

`cd /www/server/panel && btpython tools.py panel 新密码`
