## 一、安装jdk（centos）

```
// 安装jdk1.8
yum -y install java-1.8.0-openjdk*

// 安装jdk11
sudo yum install java-11-openjdk-devel
```

## 二、管理端部署，刷新页面后404，nginx中添加

```
location / {
  try_files $uri $uri/ /index.html;
}
```

## 三、jenkins

### 3.1、构建后启动项目失败
构建后执行`.sh`脚本启动项目失败

在`shell`命令中加入 
```
BUILD_ID=DONTKILLME
```

### 3.2、启动前，设置环境变量

```
export JENKINS_HOME=/data/jenkins/data
```
