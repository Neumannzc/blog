## 一、对象转 `json`

```javascript
JSON.stringify(str)
```

## 二、`json` 转对象

```javascript
JSON.parse(json)
```

## 三、数据存储到本地

```javascript
localStorage.setItem( key , value )
```

## 四、从本地中获取数据

```javascript
localStorage.getItem( key )
```
