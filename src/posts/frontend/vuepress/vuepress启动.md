vuepress 的启动

<!-- more -->

1. 安装 `curl`
```shell
sudo apt install curl
```

2. 安装 `pnpm`
```shell
npm install -g pnpm
```

3. 安装依赖
```shell
pnpm install
```

4. 启动
```shell
pnpm run docs:dev
```
