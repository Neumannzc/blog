## 一、查看是否安装

```javascript
# 如查看是否安装yarn
yarn --version
```

## 二、安装 `less`

```javascript
yarn add less less-loader -D 
或者
npm i less less-loader -D
```

## 三、安装 `axios`

```javascript
yarn add axios
或者
npm i axios
或者
pnpm add axios
```

## 四、安装 `vuex`

```javascript
yarn add vuex@3 

或者

npm i vuex@3
```

## 五、安装 `json-server`

```javascript
yarn global add json-server

或者

npm i json-server  -g
```

## 六、安装 `pnpm`

```javascript
npm install -g pnpm
```

## 七、安装`yarn`

```javascript
npm install -g yarn
```
